Create Table from plain types¶

Arrow allows fast zero copy creation of arrow arrays from numpy and pandas arrays and series, but it’s also possible to create Arrow arrays and Tables from plain Python structures.

the pyarrow.table() function allows creation of Tables from nearly all supported inputs, including plain python objects

import pyarrow as pa

table = pa.table({
    "col1": [1, 2, 3, 4, 5],
    "col2": ["a", "b", "c", "d", "e"]
})

print(table)

pyarrow.Table
col1: int64
col2: string

Note

All values provided in the dictionary will be passed to pyarrow.array() for conversion to Arrow arrays, and will benefit from zero copy behaviour when possible.

Store Categorical Data¶

Arrow provides the pyarrow.DictionaryArray type to represent in memory categorical data without the cost of storing and repeating the categories over and over when they consist of some expensive types like text.

If you have an array containing repeated categorical data, it is possible to convert it to a pyarrow.DictionaryArray using pyarrow.Array.dictionary_encode()

arr = pa.array(["red", "green", "blue", "blue", "green", "red"])

categorical = arr.dictionary_encode()
print(categorical)

...
-- dictionary:
  [
    "red",
    "green",
    "blue"
  ]
-- indices:
  [
    0,
    1,
    2,
    2,
    1,
    0
  ]

If you already know the categories and indices, you can skip the encode step and directly create the DictionaryArray using pyarrow.DictionaryArray.from_arrays()

categorical = pa.DictionaryArray.from_arrays(
    indices=[0, 1, 2, 2, 1, 0],
    dictionary=["red", "green", "blue"]
)
print(categorical)

...
-- dictionary:
  [
    "red",
    "green",
    "blue"
  ]
-- indices:
  [
    0,
    1,
    2,
    2,
    1,
    0
  ]
